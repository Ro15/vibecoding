# Presentation — AI-Driven Delivery: Three Platforms, One Architect

The submission deck for the Graduate Vibe Coding Challenge.

- **`deck.html`** — polished, self-contained HTML slide deck (glassmorphism, keyboard/swipe
  nav, dark/light toggle) with embedded live-dashboard screenshots. Open directly in any
  browser, or view the published version:
  https://claude.ai/code/artifact/ca0b355c-d0e2-47b8-aadf-fcfc93ce4412
- **`deck.md`** — the same 14 slides as a Marp Markdown deck (the format the challenge
  accepts). Render to PPTX/PDF:
  ```bash
  npx @marp-team/marp-cli docs/presentation/deck.md -o deck.pptx   # or --pdf
  ```

Navigate the HTML deck with **← / →** (or swipe on touch); **◐** toggles the theme.
